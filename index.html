键盘操作

点击节点选中，直接在图上编辑，无需填表单

  • 移动视角普通拖拽画布
  • 拖拽节点 / 家族按住 Space + 拖拽
  • 关联两个节点Shift+点击 / L
  • 节点归入家族选中家族 · Shift+点击 目标
  • 编辑关系类型Enter / 双击连线
  • 删除关系选中连线 · Del
  • 创建子节点Tab
  • 创建同级节点Enter
  • 编辑节点名称Enter / 双击
  • 删除节点Del
  • 撤销 / 重做⌘Z / ⌘⇧Z
  • 取消编辑Esc

编辑中按 Tab / Enter 会先保存当前内容，再创建下一个节点
